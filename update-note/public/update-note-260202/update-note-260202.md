# Update Note - 2026-02-02

## New Features

- **사내 대화방 그룹** 기능이 추가되었습니다.
  - 사내 대화방을 만들고 동료를 초대할 수 있습니다.
  - 사내 대화방이 그룹으로 묶여 표시됩니다.
  - 사내 일반 대화방은 나가기가 제한됩니다.

## Improvements

- 확장자가 없는 파일 업로드는 보안 및 안정성을 위해 제한됩니다.

## Bug Fixes

- 게스트가 대화방 참여자 프로필을 볼 때 편집 가능해 보이던 표시를 수정했습니다.
- 대화방 목록에서 대화방을 바꿨을 때 마지막 대화방 정보가 갱신되지 않던 문제를 수정했습니다.
- 멘션 시 @@가 두 번 보이던 문제를 수정했습니다.
- 멘션 입력 시 대상자가 바로 반영되지 않던 문제를 수정했습니다.
