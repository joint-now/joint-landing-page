# Update Note - 2025-07-15

## New Features

- **최근 메시지 알림**이 게스트 사용자에게도 제공됩니다.

## Improvements

- 알림이 많을 때 요약해 표시하도록 개선했습니다.
- 채널 목록 표시 순서를 개선했습니다.
- 긴 이름, 메모, 채널 설명 등이 잘리지 않고 보이도록 표시 방식을 개선했습니다.
- 거래처·팀 관리 페이지가 다양한 해상도에 맞게 표시됩니다.

## Bug Fixes

- 거래처 대화방에서 나가기를 하면 목록에 바로 반영되도록 수정했습니다.
- [PC] 채널 리스트 크기 조절, 대화방 만들기 모달 문구, 채널 미리보기 이미지 등을 수정했습니다.
- 초대 모달·알림·게스트 아바타 표시 등 일부 UI를 개선했습니다.
