Update Note

[Joint Update] 25.10.20 업데이트

2025년 10월 20일

✨ New Features

  • [모바일] 홈 화면 개편이 완료되었습니다.
    • 받은 초대를 플로팅으로 확인할 수 있습니다.
    • 받은 초대 수락·거절 전용 페이지가 있습니다.
  • 채널을 찾을 수 없을 때 안내하는 에러 페이지가 추가되었습니다.

⚡ Improvements

  • [모바일] 체크 아이콘을 추가·변경했습니다.
  • 알림·스레드 에디터 상단 여백을 조정했습니다.
  • [모바일] 채널 링크 미리보기, 탭 하이라이트, 스레드 버튼 등 인터랙션을 정리했습니다.
  • 모바일에서 불필요한 호버 효과를 제거했습니다.

🐛 Bug Fixes

  • [모바일] 첨부 이미지 섬네일 가운데 정렬, 챙겨야 할 메시지 탭 디자인, 채널 바텀시트 등을 개선했습니다.
  • 스레드 내 활동내역, 거래처 태그·게스트 프로필 표시, 시간 표시 형식 등을 수정했습니다.
  • 팀 관리 화면 닫기 버튼 크기·색상을 조정했습니다.
